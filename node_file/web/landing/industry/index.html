IoT장비 특수 기능

내용.

영상처리

컴퓨터에 의한 디지털 영상 처리는 메모리 장치의 급속한 발전으로 우주 관측, 의료용, 사진 분석, 디자인, 다양한 패턴 인식 등 많은 응용 분야에서 사용되고 있습니다. 영상 입력 장치로는 TV 카메라, CCD 카메라 등이 사용되고 있으며, 카피 하드, 소프트웨어 등 다양한 출력 장치가 사용되고 있으며, 컴퓨터로 디지털화 된 이미지를 처리하는 특수 기술도 개발 중입니다.

장애물탐지

장애물 감지 시스템을 도입하기 위해 Arduino 초음파 센서와 영상 처리 기술을 이용하여 구현했습니다. 그리고 라즈베리를 사용하여 탈 중앙화를 실현하기 위해 딥 러닝을 진행했습니다.

데이터 전송

현대 정보 사회의 기반이되는 데이터 통신은 컴퓨터 기반 정보 처리 기술과 컴퓨터 네트워크 관련 데이터 전송 기술의 통합 된 형태입니다. 데이터 통신은 데이터 전송 기술과 데이터 처리 기능의 조합으로 이해 될 수 있습니다.
